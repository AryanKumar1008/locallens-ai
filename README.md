# LocalLens AI

## Problem
Tourists often receive generic travel recommendations and miss authentic local experiences.

## Solution
LocalLens AI generates personalized travel itineraries using Gemini AI and provides an AI-powered local guide chatbot.

## Features
- AI itinerary generation
- Ask a Local chatbot
- Hidden gems
- Cultural storytelling
- Local food recommendations
- Responsible travel tips
- Responsive UI

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Gemini 2.5 Flash API
- Vercel

## Architecture
User → Next.js API Routes → Gemini API → JSON Response → UI

## Future Scope
- Maps integration
- Weather
- Multi-language support
- Booking integration
